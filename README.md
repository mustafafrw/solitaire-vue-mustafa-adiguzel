# Solitaire Project with Vue.js

Demo: https://agitated-perlman-ed8c22.netlify.app/#/

## Technologies
- Vue
- Vuex
- Trendyol Grace https://github.com/Trendyol/grace

## Features
- Two game option - Reverse and normal solitaire
- Multi Drag
- Getting hints
